# Spyro: A Hero's Tail Archipelago
An implementation of the Archipelago randomizer for Spyro: A Hero's Tail for Playstation 2 and GameCube.

## What is randomized?

This list is non exhaustive, and they may not be implemented in future releases.

- [ ] Dark Gems
- [ ] Light Gems
- [ ] Dragon Eggs
- [ ] Elemental Breaths
  - [ ] + Fire Breath
- [ ] Elder abilities
- [ ] Mini game rewards
- [ ] Shop gem costs
- [ ] Light Gem door costs
- [ ] Charge, Swim and Glide
